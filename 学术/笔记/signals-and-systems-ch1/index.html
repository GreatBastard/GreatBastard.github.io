Invertibility and Inverse Systems

A system is said to be invertible if distinct inputs lead to distinct outputs. If a system is invertible, then an inverse system exists that, when cascaded with the original system, yields an output $w[n]$ equal to the input $x[n]$ to the first system.

Causality

A system is causal if the output at any time depends on values of the input at only the present and past times. Such a system is often referred to as being non-anticipative, as the system output does not anticipate future values of the input. Consequently, if two inputs to a causal system are identical up to some point in time $t_0$ or $n_0$ , the corresponding outputs must also be equal up to this same time.

Stability

The bounded-input bounded-output (BIBO) stability implies that for every bounded input, $u(t):|u(t)|<M_1<\infty$, the output of the system stays bounded, that is, $y(t):|y(t)|<M_2<∞$.

Time Invariance

A system is time invariant if the behavior and characteristics of the system are fixed over time.
Specifically, a system is time invariant if a time shift in the input signal results in an identical time shift in the output signal. That is, if $y[n]$ is the output of a discrete-time, time-invariant system when $x[n]$ is the input,
then $y[n-n_0]$ is the output when $x[n-n_0]$ is applied. In continuous time with $y(t)$ the output corresponding to the input $x(t)$, a time-invariant system will have $y(t-t_0)$ as the
output when $x (t-t_0)$ is the input.

Linearity

Let $y_1(t)$ be the response of a continuoustime system to an input $x_1(t)$, and let $y_2(t)$ be the output corresponding to the input $x_2(t)$.Then the system is linear if:

  1. Additivity or superposition property: The response to $x_1(t)+x_2(t)$ is $y_1(t)+y_2(t)$.
  2. Scaling or homogeneity property: The response to $ax_1(t)$ is $ay_1(t)$, where $a$ is any complex constant.
